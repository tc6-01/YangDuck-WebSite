# 🐤 YangDuck Website

yduck 项目官网 — 单页 Landing Page，中英双语。

## 开发

```bash
npm install
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看。

## 部署

部署到 Vercel，Root Directory 设为 `yduck-website`。
